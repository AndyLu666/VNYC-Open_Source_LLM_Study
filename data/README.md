# Data Directory

This directory contains analysis-ready data used by the manuscript.

Primary folders:

- `derived/database/19_jcdl_revision/`: main JCDL analysis outputs.
- `derived/database/20_jcdl_validity_strengthening/`: audit intervals, construct-validity tables, threshold rationale, and failure-mode cases.
- `derived/database/16_www_revision/`: one minimal query-sensitivity support table retained for scope checks.
- `derived/database/18_www_reviewer_requested_validations/`: artifact-role audit support used for Figure 2.

See `../docs/DATA_DICTIONARY.md` for field-level details.
