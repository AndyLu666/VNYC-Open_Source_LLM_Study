# Data

This directory contains the analysis-ready data tables used by the paper. The files are organized by analysis stage rather than by raw collection source.

## Directory Map

- `main_analysis/`: repository metadata completeness, paper-level curatability scores, provenance tiers, release-package evidence, paper-repository alignment, and claim-support ledgers.
- `validity_checks/`: audit confidence intervals, construct-validity mappings, threshold rationales, schema review cases, and curated failure modes.
- `sensitivity_checks/`: query-bucket sensitivity support retained to document the fixed query-based repository snapshot.
- `audit_support/`: balanced repository-role audit support used by the manuscript figures.

The field-level description is in `../docs/DATA_DICTIONARY.md`.
