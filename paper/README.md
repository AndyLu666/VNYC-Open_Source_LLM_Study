# Paper Source

This directory contains an anonymous ACM/JCDL manuscript source snapshot and the final figures/tables used by the submitted paper.

To compile:

```bash
latexmk -pdf -interaction=nonstopmode main.tex
```

The paper source is included for reviewer convenience. The main reproducibility entry points are the analysis CSVs under `../data/` and the figure/table generation script under `../scripts/`.
