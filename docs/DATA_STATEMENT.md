# Data Statement

The released tables are derived from public model-hub records, public scholarly metadata, and public paper/repository traces. The artifact is scoped to compact and derived open LLM artifacts and uses analysis-ready tables rather than raw platform snapshots.

## Included

- Derived repository metadata indicators.
- Derived paper-level curatability indicators.
- Paper-repository alignment edges.
- Provenance-evidence tiers.
- Release-package evidence tiers.
- Expert-audit and construct-validity summaries.

## Excluded

- API credentials and local environment files.
- Raw LLM API inputs/outputs.
- Downloaded PDFs and OCR/GROBID intermediate files.
- Full raw Hugging Face/source snapshots that are large and not necessary for reviewer inspection.
- Internal notes that are not part of the submitted artifact.

## Licensing and Source Terms

Code in this artifact is intended for research reuse under the repository license. The tabular data are derived from public records and should be used for research and review purposes with appropriate citation to the paper and respect for the terms of the original public sources.
